# Crypto Price Fetcher

My very first Python project!

This script uses the ccxt library to fetch live BTC/USDT price data from Binance.

Goal: Learn crypto APIs, data fetching, and quant trading basics.

Inspired by exploring market making and algorithmic trading.

Next steps: Add backtesting, order book analysis, and more!
